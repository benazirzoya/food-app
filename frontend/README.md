# Foodie - Frontend Application 🍕🍔

This is the React-based frontend client for the **Foodie** food delivery clone application, developed by **Benazir Zoya**.

---

## 📸 Mockups

For visual screenshots of the website, please see the [Root README.md](../README.md).

---

## 🛠️ Features Implemented

1. **Vibrant Glassmorphic UI**: High-fidelity dark mode with neon red and gold navigation highlights.
2. **Interactive 3D Chef**: Canvas-rendered walking 3D Chef model reacting dynamically to nav item hovers.
3. **End-to-End User Profiles**: Native file uploader with preview thumbnail for profile avatars (saved via Base64).
4. **Auto-Sliding Reviews Carousel**: Submit reviews with custom or preset food images, which immediately populate and slide in a custom auto-scrolling timeline.
5. **Simulated Payments**: Custom simulated Razorpay Checkout portal with Secure OTP banking codes.
6. **Mobile Viewport Zoom Prevention**: Custom hooks that block browsers from auto-zooming input fields.

---

## 🚀 Installation & Local Development

1. Install frontend packages:
   ```bash
   npm install
   ```
2. Start the React development server:
   ```bash
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
